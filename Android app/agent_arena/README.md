# 🤼 Agent Arena

**Multi-Agent AI Debate Platform** — powered by **LangGraph** + **Gemini API**.

Two AI agents, **Rambahaur** (FOR) and **Shaam Bahadur** (AGAINST), debate any topic you throw at them in a real-time Street-Fighter-style arena. Every response is generated by Google's Gemini LLM, orchestrated through a LangGraph state machine.

---

## 🎭 The Agents

| Agent | Alias | Side | Personality | Tone |
|-------|-------|------|-------------|------|
| **Rambahaur** 🦁 | Pro | FOR the topic | Cocky, lion-hearted, roaring confidence | `[aggressive]` or `[serious]` |
| **Shaam Bahadur** 🌙 | Con | AGAINST the topic | Slick, snarky, regal, witty comebacks | `[sarcastic]` or `[aggressive]` |

Each agent ends their response with a tone tag (`[sarcastic]`, `[serious]`, `[aggressive]`) which triggers different sound effects and HP damage in the arena.

## ⚙️ Architecture

### LangGraph Debate Pipeline

```
User enters topic → /api/debate/start → Debate created in Firestore
                                        ↓
                    ┌─────────────────────────────────────┐
                    │  POST /api/debate/next-round         │
                    │  (called repeatedly by Flutter)       │
                    └──────────┬──────────────────────────┘
                               ↓
                    ┌─────────────────────┐
                    │  PRO node            │  Rambahaur generates argument
                    │  (generate_pro_arg)  │  + tone tag
                    └──────────┬──────────┘
                               ↓
                    ┌─────────────────────┐
                    │  CON node            │  Shaam Bahadur counters
                    │  (generate_con_arg)  │  + tone tag
                    └──────────┬──────────┘
                               ↓
                    ┌─────────────────────┐
                    │  JUDGE node          │  Scores both out of 10
                    │  (judge_round)       │  + commentary
                    └──────────┬──────────┘
                               ↓
                    Round saved → if max_rounds reached →
                    winner declared (total score)
```

### Tech Stack

| Layer | Tech |
|-------|------|
| **Frontend** | Flutter + Riverpod + GoRouter |
| **Backend** | FastAPI (Python) |
| **LLM** | Google Gemini API (`gemini-2.0-flash-lite`) |
| **State machine** | LangGraph |
| **Database** | Firebase Firestore |
| **Audio** | `audioplayers` package |

## ✨ Features

- **Real-time fight arena** — animated sprites, health bars, speech bubbles with typewriter effect
- **Tone-based sound FX** — aggressive hits, sarcastic laughs, serious thuds — mapped from the AI's tone tag
- **Dynamic display timing** — speech bubbles stay on screen proportional to response length
- **Auto-navigate home** — 2 seconds after winner is announced
- **Debate history** — view past debates, judge scores, and round-by-round breakdown
- **Chaos Mode** — custom agent names and topics for unscripted debates
- **Multi-round scoring** — judge scores each round; cumulative total decides the winner (ties broken randomly)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK ^3.11
- Python 3.11+
- A Gemini API key from [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
- Firebase project (for Firestore)

### 1. Backend Setup

```bash
cd backend
cp .env.example .env
```

Edit `.env` and fill in:
```
GEMINI_API_KEY=your_actual_gemini_api_key
GEMINI_MODEL=gemini-2.0-flash-lite
FIREBASE_PROJECT_ID=your_firebase_project
FIREBASE_PRIVATE_KEY="..."
FIREBASE_CLIENT_EMAIL=...
```

Install dependencies and run:
```bash
pip install -r requirements.txt
python -m uvicorn backend.main:app --reload
```

### 2. Flutter App

```bash
cd Android\ app/agent_arena
flutter pub get
flutter run
```

### 3. Build APK

```bash
flutter build apk --release
# Output: build/app/outputs/flutter-apk/app-release.apk
```

## 🔌 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/debate/start` | Create a new debate |
| `GET` | `/api/debate/{id}` | Get debate status |
| `POST` | `/api/debate/next-round` | Run one round (pro → con → judge) |
| `POST` | `/api/debate/vote` | Cast a vote |
| `GET` | `/api/debate/{id}/results` | Get vote results |
| `GET` | `/api/history` | List past debates |
| `GET` | `/api/history/{id}` | Get debate detail |
